Diagram lingkaran di samping menunjukkan file yang terdapat di dalam flashdisk milik reta yang berkapasitas 4 GB (setara dengan 4.000 MB). flashdisk tersebut diisi dengan file musik, foto, data buku ajar matematika, dan data lainnya.

A.    Jika Reta ingin menambahkan file data buku ajar baru yang berkapasitas 750 MB, apakah kapasitas flashdisk milik reta masih mencukupi?, jelaskan!. B.    Jika Reta tidak ingin menghapus file foto, file data buku ajar, dan file data lainnya di flashdisknya, berapa persen dari keseluruhan file musik yang harus dihapus agar data buku ajar baru dapat ditambahkan ke dalam flashdisk?

Statistika adalah ilmu (metode ilmiah) yang mempelajari cara-cara mengumpulkan, menyusun, menyajikan, dan menganalisis data serta cara mengambil kesimpulan yang logis sehingga dapat diambil keputusan yang akurat.

Pembahasan

Perhatikan gambar diagram lingkaran yang ada pada lampiran.

A.  Menambahkan file data buku ajar baru berkapasitas 750 MB

Kapasitas kosong = [tex]\frac{10}{100}[/tex] × 4000 MB

                              = 400 MB

Ternyata flasdisk Reta tidak mencukupi apabila diisi dengan file data buku ajar baru berkapasitas 750 MB. Pada kapasitas flasdisk ruang kosong hanya tersedia 10% atau 400 MB, sehingga tidak dapat mencukupi file yg diisikan.

B.  Persen data yang dihapus dari keseluruhan file musik

Kekurangan data (dihapus) = kapasitas file baru - kapsitas ruang kosong

                                             = 750 MB - 400 MB

                                             = 350 MB

Kapasitas file musik = [tex]\frac{20}{100}[/tex] × 4000 MB

                                 = 800 MB

% file data yg dihapus dari keseluruhan file musik

= [tex]\frac{data~ dihapus}{file~ musik}[/tex] × 100 %

= [tex]\frac{350}{800}[/tex] × 100 %

= 43,75 %

Jadi file musik yang harus dihapus supaya mencukupi file baru adalah 43,75 % dari file musik

5. seorang siswa mempunyai flash disk dengan kapasitas penyimpanan sebesar 8 Gb, di dalam flashdisk tersebut diisi beberapa data yaitu 100 foto yang masing-masing foto berukuran 20 MB dan file video sebesar 1.450 MB serta 1 file aplikasi sebesar 4,12 GB. berapakah sisa kapasitas penyimpanan flashdisk siswa tersebut?pke cara ya.​


Jawaban:

3,701,518 MB.

Penjelasan:

Jumlah kapasitas penyimpanan yang digunakan oleh foto adalah 100 foto x 20 MB/foto = 2.000 MB.

Sedangkan jumlah kapasitas penyimpanan yang digunakan oleh video adalah 1.450 MB.

Total kapasitas penyimpanan yang digunakan adalah 2.000 MB + 1.450 MB + 4,12 GB = 7.570 MB + 4,12 GB = 7.570 MB + 4,290,912 MB = 4,298,482 MB.

Sisa kapasitas penyimpanan flashdisk siswa tersebut adalah 8 GB - 4,298,482 MB = 3,701,518 MB.

Jawaban:

Untuk mengetahui sisa kapasitas penyimpanan flash disk siswa tersebut, pertama-tama harus menghitung jumlah ukuran total dari semua file yang ada di dalam flash disk tersebut.

Jumlah ukuran total dari foto-foto tersebut adalah 100 foto x 20 MB/foto = 2.000 MB

Jumlah ukuran total dari semua file yang ada di flash disk tersebut adalah 2.000 MB + 1.450 MB + 4.12 GB = 7.570 MB + 4.12 GB = 4,93 GB.

Kemudian, untuk mengetahui sisa kapasitas penyimpanan flash disk tersebut, bisa mengurangi jumlah ukuran total file dari kapasitas penyimpanan flash disk yaitu 8 GB - 4,93 GB = 3,07 GB. Jadi, sisa kapasitas penyimpanan flash disk siswa tersebut adalah 3,07 GB.

Jika diinginkan, bisa juga mengubah jumlah ukuran total file yang ada di flash disk tersebut ke dalam satuan MB, sehingga sisa kapasitas penyimpanan flash disk siswa tersebut dapat dinyatakan dalam satuan MB juga.

Jumlah ukuran total file yang ada di flash disk tersebut adalah 4,93 GB = 4.930 MB.

Sisa kapasitas penyimpanan flash disk siswa tersebut adalah 8.000 MB - 4.930 MB = 3.070 MB.

8. Berapa jumlah lagu mp3 (masing masing lagu 3 mb) yang dapat di simpan pada flashdisk 8 gb


Terdapat sebuah diska lepas (flash disk). Kapasitas diska tersebut adalah 8 GB. Misalkan terdapat lagu yang masing-masingnya berukuran 3 MB. Diska lepas tersebut dapat menyimpan 2730 lagu.

Penjelasan dengan langkah-langkah

Diketahui:

Kapasitas diska lepas = 8 GB

Ukuran lagu = 3 MB

Ditanya: banyaknya lagu yang dapat disimpan dalam diska lepas

Jawab:

Konversi satuan

Satuan dari kedua besaran yag diberikan berlainan. Salah satunya akan diubah sehingga satuannya sama. Agar proses perhitungannya masih melibatkan bilangan bulat, ubah nilai kapasitas diska lepas ke satuan MB. Ingat bahwa: 1 GB = 1024 MB.

Kapasitas diska lepas = 8 GB = 8×1024 = 8192 MB

Banyaknya lagu yang dapat disimpan

Banyaknya lagu = kapasitas diska lepas/ukuran lagu

= 8192/3

≈ 2730,67

Karena banyaknya lagu harus dalam bilangan bulat, maka dibulatkan ke bawah (apabila ke atas, akan melampaui kapasitas diska lepas). Jadi, lagu yang dapat disimpan ke dalam diska lepas tersebut sebanyak 2730 lagu.

30. siapa penemu data seperti KB,MB,GB?​


Jawaban:

KB, MB, GB, dan TB sering sahabat temukan dalam bidang komputasi seperti dalam perhitungan besarnya Kuota internet, kapasitas RAM (Random Access Memory), dan ruang penyimpanan Hard disk drive. Untuk mengetahui seberapa besar / banyaknya hal-hal yang berhubungan dengan data komputer tersebut, maka pada kesempatan kali ini saya akan memberikan gambaran dan mengenalkan kepada sahabat yang masih mengalami kesulitan dalam mengenali dan memahami kilobyte, megabyte, dan gigabyte.

Pengertian KB, MB, GB, dan TB

KB, MB, GB, dan TB adalah satuan yang digunakan dalam menghitung data digital. Seperti yang sempat saya singgung sebelumnya bahwa istilah tersebut banyak digunakan di bidang komputansi salah satunya adalah dalam menghitung kuota data internet. Dengan mengenal satuan-satuan tersebut, maka saya berharap bahwa sahabat pembaca blog ini dapat memiliki gambaran tentang seberapa banyak kuota internet ataupun besarnya kapasitas Hard disk dan RAM dari sebuah perangkat digital yang sahabat miliki.

Urutan KB, MB, GB, dan TB

Berikut akan saya urutkan satuan data tersebut dari yang paling kecil hingga yang paling besar. [1] Byte biasa disingkat dengan B, [2] Kilobyte biasa disingkat dengan KB, [3] Megabyte biasa disingkat dengan MB, [4] Gigabyte biasa disingkat dengan GB, [5] Terabyte biasa disingkat dengan TB, [6] Petabyte biasa disingkat PB, [7] Exabyte biasa disingkat EB, [8] Zettabyte biasa disingkat ZB dan, [9] Yottabyte biasa disingkat YB.

Cara Menghitung & Mengkonversi KB, MB, GB, dan TB

Bila tadi sahabat sudah mengetahui urutan dari satuan tersebut, pastinya sahabat sudah mengetahui mana yang lebih besar dan mana yang lebih kecil dari satuan-satuan tersebut. Pada kesempatan kali ini saya akan membahas tentang cara menghitung dan mengkonversinya. Konversi sendiri bisa berupa perhitungan Biner dan perhitungan Desimal. Bila sahabat ingin mudah menghitungnya maka saya sarankan menggunakan konversi desimal. Berikut adalah perhitungan dan konversi dari Byte, Kilobyte, Megabyte, Gigabyte, dan Terabyte.

1 KB (Kilobyte) berapa B (Byte)?

Bila ingin menghitungnya dengan menggunakan konversi perhitungan biner, maka 1KB = 1024B, sedangkan bila menggunakan konversi perhitungan desimal maka 1KB = 1000MB.

1 MB (Megabyte) berapa KB (Kilobyte)?

Dengan menggunakan konversi perhitungan biner, maka 1MB = 1024KB dan dengan menggunakan konversi perhitungan desimal maka 1MB = 1000KB

1 GB (Gigabyte) berapa MB (Megabyte)?

Dengan menggunakan konversi perhitungan biner, maka 1GB = 1024MB dan dengan menggunakan konversi perhitungan desimal maka 1 GB = 1000MB

1 TB (Terabyte) berapa GB (Gigabyte)?

Dengan menggunakan konversi perhitungan biner, maka 1TB = 1024GB dan dengan menggunakan konversi perhitungan desimal maka 1 TB = 1000GB

Memahami besarnya KB, MB, GB, dan TB

Supaya sahabat bisa memahami dan bisa menjawab pertanyaan: Sebenarnya besar atau tidak ketika memiliki kuota internet 1GB? Jawabannya adalah Relatif. Kenapa relatif? Karena sesuai kebutuhan sahabat. Untuk lebih jelasnya silakan sahabat perhatikan contoh di bawah ini:

Saya memiliki kuota 1 GB kemudian saya gunakan untuk streaming youtube, maka dalam hitungan antara 1 – 3 Jam kuota tersebut sudah habis karena ukuran file video cukup besar. Sebagai gambaran saja File Video dengan durasi 2 Jam memiliki ukuran file sekitar 1 GB. Dengan begitu kuota 1 GB akan cepat habis dalam hitungan jam.

Ketika saya memiliki kuota 1 GB dan saya gunakan hanya untuk baca berita menggunakan web browser dan untuk kirim pesan saja menggunakan aplikasi whatsapp, line, dan BBM. Maka kuota tersebut akan bisa bertahan sampai satu bulan. Hal ini dikarenakan konsumsi untuk mengakses berita dan melakukan kirim pesan tidak memerlukan kuota yang banyak. Meski demikian, tidak menutup kemungkinan kuota tersebut juga bisa habis dalam hitungan hari kalau pesan yang dikirim dan diterima berupa Video yang ukurannya besar-besar.

Bila sahabat sudah mengerti, mengenali, bisa menghitung, dan sudah memahami tentang ukuran KB, MB, GB, dan TB maka saya yakin bahwa sahabat tidak akan lagi kesulitan untuk menilai apakah kuota internet, kapasitas RAM, Memory Internal, dan Hard disk suatu perangkat besar atau kecil. Semoga membantu dan bermanfaat…
