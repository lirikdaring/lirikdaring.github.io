5. Harga sebuah Laptop adalah 5 kali harga sebuah monitor. Harga 5 buah monitor dan 4 buahlaptop adalah Rp. 20.000.000. Berapakah harga sebuah laptop?Bantuin sekarang pls :'( ​

Penjelasan dengan langkah-langkah:

anggap laptop = x dan monitor = y

• buat model matematika

laptop = 5•monitor

5monitor + 4laptop = 20.000.000

maka persamaannya :

x = 5y (persamaan I)

5y + 4x = 20.000.000 (persamaan II)

• substitusi x persamaan I ke persamaan II

5y + 4x = 20.000.000

5y + 4(5y) = 20.000.000

5y + 20y = 20.000.000

25y = 20.000.000

y = 20.000.000/25

y = 800.000

• substitusi y ke x persamaan I

x = 5y

x = 5(800.000)

x = 4.000.000
