11. tolong dibantu harga 12 cd tipe A dan 8 cd tipe B adalah rp.142.000,00 dari. jika harga cd tipe B lebih mahal rp.1.500,00 dari cd tipe A tentukanlah harga untuk cd tipe A. dan 15 cd tipe B!


Misal cd tipe A=x
tipe B = y
maka berlaku hubungan
12x + 8y = 142000 ... sederhanakan dibagi 4 menjadi
3x + 2y = 35500 .... Persamaan (1)
y = x + 1500 ... Persamaan (2)

Substitusi persamaan (2) ke (1)
3x + 2y = 35500
3x + 2. (x + 1500) = 35500
3x + 2x + 3000 = 35500
5x = 35500 - 3000
5x = 32500
x = 32500 : 5
x = 6500

Sub x= 6500 ke Pers (2)
y = x + 1500
= 6500 + 1500
= 8000

Jadi harga cd tipe A adalah Rp. 6500, - dan cd tipe B adalah Rp. 8000,-

40. Harga 4 CD Tipe A Dan 2 CD Tipe B Adalah Rp. 50.000 Diketahui Harga CD Tipe B Lebih Mahal Rp. 1000 Dari Harga Sebuah CD Tipe A. Tentukan Biaya Yang Harus Dibayarkan Oleh Joko Jika Ia Membeli 5 CD Tipe A Dan 6 CD Tipe B​


Jawaabn :

Rp. 94.000

Penjelasan dengan langkah-langkah:

misal CD tipe A itu x

misal CD tipe B itu y

diketahui : y lebih mahal 1000 dari x

4x + 2y = 50.000

karena y lebih mahal 1000 dari x

maka y dapat ditulis x + 1000

persamaan menjadi :

4x + 2(x + 1000) = 50000

4x + 2x + 2000 = 50000

6x = 48000

x = 8000

maka nilai CD tipe A = 8000

sekarang karena sudah di ketahui x-nya, di bilang di situ kalo y lebih mahal 1000 dari x

jadi y = x + 1000

y = 9000

maka nilai CD tipe B = 9000

yang ditanya adalah:

5 CD tipe A + 6 CD tipe B

subtitusi menjadi

5 × 8000 + 6 × 9000

ingat kali di buat duluan

40000 + 54000

jadi RP. 94.000

jadikan yang terbaik!
