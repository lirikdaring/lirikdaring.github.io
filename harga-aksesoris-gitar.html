23. pak adi membeli sepeda dengan harga Rp450.000 kemudian sepeda tersebut diberi aksesoris seharga Rp100.00.jika sepeda tersebut dijual dengan harga Rp600.000, berapa presentase keuntungannya?​


Jawab:

Persentase untung = 0,09%

Penjelasan dengan langkah-langkah:

Diketahui:

Harga beli = Rp450.000,-

Diberi aksesoris = Rp100.000,-

---> Jadi harga total = 450.000 + 100.000 = 550.000

Harga jual = Rp600.000,-

Ditanya:

Persentase untung = .....?

Jawab:

Keuntungan = harga jual - harga beli

Keuntungan = Rp600.000 - Rp550.000

Keuntungan = Rp50.000,00

Persentase untung = (keuntungan : harga beli) x 100%

Persentase untung = (Rp50.000 : Rp550.000) x 100%

Persentase untung = [tex]\frac{1}{11}[/tex] x 100%

Persentase untung = 0,09%

Penjelasan dengan langkah-langkah:

diketahui :

harga beli = Rp450.000 + Rp100.000 = Rp550.000

harga jual = Rp600.000

ditanya : berapa persentase keuntugan?

jawab :

jumlah untung

= harga jual - harga beli

= Rp600.000 - Rp550.000

= Rp50.000

persentase keuntugan

= untung/harga beli × 100

= Rp50.000/Rp550.000 × 100

= 0,09%

jadi, persentase untung = 0,09%
